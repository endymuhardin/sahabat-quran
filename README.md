# Yayasan Sahabat Quran #

Aplikasi Manajemen Operasional Yayasan Sahabat Quran

## Daftar Fitur ##

### Admin ###

* Data Pengajar
* Data Bank dan Rekening
* Data Kurikulum
* Data Ruangan
* Data Kelas
* Plotting Kurikulum ke Kelas
* Plotting Kelas ke Pengajar
* Plotting Kelas ke Ruangan
* Jadwal Belajar
* Jadwal Ujian
* Soal Ujian
* Parameter penilaian ujian / bobot penilaian
* Event

### Finance ###

* Setting Nilai Iuran
* Generate Tagihan Iuran
* Pembayaran Iuran (Manual dan VA)
* Infaq non Iuran

### Peserta ###

* Registrasi
* Lihat Tagihan
* Pembayaran Iuran
* Jadwal Belajar
* Input Mutabaah (catatan kemajuan pelajaran)
* Jadwal Event
* Input Presensi Kelas
* Input Presensi Event
* Laporan Hasil Belajar

### Pengajar ###

* Daftar Kelas yang dipegang
* Jadwal Mengajar
* Start sesi belajar
* Input presensi peserta
* Finish sesi belajar
* Input berita acara sesi pelajaran
* Lihat mutabaah peserta
* Start sesi ujian praktek
* Input nilai ujian praktek
* Finish sesi ujian praktek
* Rekap sesi pengajaran
* Slip Gaji
* Rekap sesi ujian


## Technology Stack ##

* Java 21
* Maven 3
* Spring Boot 3.5
* PostgreSQL 17
* Thymeleaf
* Tailwind CSS
* TestContainer
* Jacoco Test Coverage
* Selenium WebDriver
